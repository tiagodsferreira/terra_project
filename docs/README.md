# Documentation Directory (docs)

This directory contains governance, methodological, and administrative documentation for the TERRA project. It supports transparency, traceability, and reproducibility by formally documenting project decisions, data policies, and analytical plans.

---

## Contains:

- Data Management Plan (DMP)
This folder contains a copy of Version 1.0 of the #TERRA Data Management Plan (DMP). The authoritative archived version of this document is available on Zenodo and should be cited as:
Ferreira, T., & Nunes, F. (2026). #TERRA data management plan (Version 1.0) [Data management plan]. Zenodo. https://doi.org/10.5281/zenodo.18773549

- Dissemination Strategy Guide (DSG)
Ferreira, T., & Nunes, F. (2026). #TERRA Dissemination Strategy Guide (Version 1.0) [Dissemination Strategy Guide.

- Contingency Plan (CP)
Ferreira, T., & Nunes, F. (2026). #TERRA Contingency Plan (Version 1.0) [Contingency Plan].
